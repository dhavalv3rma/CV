Book Store App

Project Link

he Book Store application is a MERN stack-based project that enables users to perform comprehensive CRUD operations on a list of books. The frontend is built with React, utilizing React Hooks for efficient state management, and Axios for seamless API integration. The backend, developed with ExpressJS, provides robust RESTful API endpoints, while MongoDB serves as the database for persisting book data. Cross-Origin Resource Sharing (CORS) is implemented to ensure secure and reliable communication between the frontend and backend. This application delivers a fully synchronized, user-friendly interface for managing books, offering features to create, read, update, and delete records efficiently.
